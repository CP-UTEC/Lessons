Force Brute

Likely, “force brute” is the first solution who someone can build for almost all problem.

But there exists anothers techniques (Divide and Conquer, Greedy, Dynamic Programming). We will return to talk about theses topics more later.

Force brute is an approach that you need to do a complete search for all search space and picking the best one. Here, we talk search space to all feasible solutions or domain through one algorithm searches. Actually, it really simple understand what happend with this technique, but this has got several optimizations depending of the problems.

A little example to apply force brute; - CodeChef Problem: Smiling Queen

#include<bits/stdc++.h>

using namespace std;

#define REP(i,a,b) for(int i=a;i<b;i++)

int main(){

  int t,s;
  cin>>t;
  string line,word,ans;
  while(t--){

    unordered_map<string,bool> dict;

    cin>>s;
    //This sentence helps us to ommit the '\n' command and not getting it for getline later
    cin.ignore();

    //Useful macro
    vector<string> stringnos;

    int cont=0;

    REP(i,0,s){

      //Get the sentence
      getline(cin,line);
      //Get the answer if she smile or not
      getline(cin,ans);

      //Create a vector to keep the sentences when the answer is "no smile"

      if(ans=="yes"){
        stringstream s(line);
        //In first moment, all words make happy to her
        while(s>>word){
          //look if the word is already true for no repetition counting
          if(!dict[word]){
            dict[word]=1;
            cont++;
          }
        }
      }
      else
        stringnos.push_back(line);
    }

    //Changing the words when she is not smile
    for(string &l:stringnos){
      stringstream s(l);
      while(s>>word){
        if(dict[word])
          cont--;
        dict[word]=0;
      }
    }
    cout<<cont<<endl;

  }

  return 0;
}
Pythagorean Triple

A Pythagorean Triple is a set of solutions in form (x,y,z) for the equation x^2 + y^2 = z^2. Each value is inside positive integers.

The smallest and best-known Pythagorean triple is (a,b,c)=(3,4,5).

Properties:

Since x,y and z, positive integers. It is possible to change the expressions as to below:

x=2mn, y=m2-n2, z=m2-n2.

where m and n are relatively-prime positive integers, and m>n.

Also, a pythagorean triple is called primitive, if gcd(x,y,z)=1.
